# 個人網站需求摘要

## 網站定位

- 朱彥勳 Patrick Zhu 的公開個人首頁，不以求職為唯一目的。
- 讓任何訪客理解本人背景、近期關注、文章觀點與工程實作。
- 核心順序：認識本人、近期關注、文章與觀點、經歷、工程專案。

## 公開內容

- 首頁：人物定位、關於我、近期關注、文章入口、精簡經歷與代表專案。
- 文章與觀點：依時間排列，支援標籤與搜尋；不先建立固定分類。
- 專案：卡片總覽與獨立內頁，保留真實圖片、技術背景及責任邊界。
- 經歷：緊湊、可展開的完整時間條列。
- 履歷：一頁式中文 PDF；英文版本完成後再加入。

## 內容邊界

- 私人 Obsidian 第二大腦與公開網站分開。
- 文章只發布已整理成熟的內容，不放示範文章。
- 不提供留言，不自動發布到 Threads 或 X。
- 團隊成果與本人負責部分必須分開，不猜測或誇大貢獻。
- 專案圖片必須由原始作品集或可追溯來源核對。

## 語言與聯絡方式

- 使用繁體中文與台灣用語。
- 網站介面同頁切換中文／英文並記住選擇。
- 文章只維持原始語言，不強制翻譯。
- 公開 Email、GitHub 與 LinkedIn。

## 維護與部署

- 專案、圖片與履歷由 `_career_data` 單一來源同步。
- 文章使用 Astro Content Collections 與 Markdown／MDX。
- GitHub Repository 保存 Astro 原始碼。
- Cloudflare Pages 由 `main` 分支自動建置 `dist`。
- 正式網址為 `yanxunzhu.com`，`www` 轉址至根網域。
- 啟用 sitemap、robots、canonical、Open Graph、結構化資料與 Cloudflare Web Analytics。
