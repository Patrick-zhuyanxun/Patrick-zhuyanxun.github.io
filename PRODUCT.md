# 朱彥勳個人網站

## 產品定位

這是一個讓任何訪客認識朱彥勳 Patrick Zhu 的公開個人首頁，不以求職轉換為唯一目的。網站記錄近期關注、經過整理的文章與觀點、工程實作和個人經歷。

## 內容順序

1. 認識朱彥勳
2. 近期關注
3. 文章與觀點
4. 學習、工作與專案經歷
5. 工程專案紀錄

## 內容原則

- 使用繁體中文與台灣用語，介面可同頁切換英文。
- 文章只維持原始語言，不強制翻譯。
- 不放示範文章、假成果或無法追溯的專案圖片。
- 團隊成果與本人負責範圍必須分開呈現。
- 私人 Obsidian 知識庫不公開，只發布整理成熟的內容。
- 不提供留言，也不自動同步發布到 Threads 或 X。

## 維護方式

- 專案與履歷由既有 `_career_data` 單一來源同步。
- 公開文章使用 `src/content/writing/` 的 Markdown 或 MDX。
- 首頁近期關注維護於 `src/data/site.ts`。
- GitHub 保存原始碼，Cloudflare Pages 自動建置並發布至 `yanxunzhu.com`。
